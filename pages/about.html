What is CSGO starter kit?

CSGO starter kit is a project for SI539, it is a non profit website that aims at providing basic knowledge for those who are interested in playing csgo. Here, you can find weapon performances and descriptions, popular weapon skins, and news and tips about CSGO. With these resources, we hope you can have a pleasant CSGO experience.

Q-Learning in Smart Traffic Light Control

In this project, we use python to simulate a two-way cross intersection with no turn and build a traffic light system. By using Q-learning algorithm and training the Q-learning model for 1,000,000 times, we obtained an optimized passing time for the traffic light control system under the condition of known car numbers on each lane.

An Embedded System of Color Extractor

In this project, we program on PIC-32 board to build a color extractor that can:

  • sense and pick color from environment
  • adjust the picked color according to usersâ€™ need
  • convert RGB values to a printable color description
  • display RGB values and the corresponding color on a screen
  • mix pigments according to printable color description, so that the final color is the same as desired one

Challenge in this project is that we write all the drivers for the components attached to the board ourselves, based on communication protocols. We also designed and build the prototype and user interface in this project.
